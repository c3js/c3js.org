Options

bindto

The element, by CSS selector, which the chart will append to. Default is '#chart'.

{ bindto: '#myContainer' }

size.width

The desired width of the chart element.

{ size: { width: 100 } }

size.height

The desired height of the chart element.

{ size: { height: 320 } }

padding.top

The padding on the top of the chart.

{ padding: { top: 0 }}

padding.right

The padding on the right of the chart.

{ padding: { right: 0 }}

padding.bottom

The padding on the bottom of the chart.

{ padding: { bottom: 0 }}

padding.left

The padding on the left of the chart.

{ padding: { left: 0 }}

color.pattern

Set custom color pattern.

color: { pattern: ['#1f77b4', '#aec7e8', '#ff7f0e', '#ffbb78', '#2ca02c', '#98df8a' ] }

interaction.enabled

Indicate if the chart should have interactions. Default is true.

interaction: { enabled: true || false }

transition.duration

Set duration of transition (in milliseconds) for chart animation. Default is 350ms.

transition: { duration: 100 }

onmouseover

Set a callback to execute when mouse enters on chart.

onmouseover: function() { ... }

onmouseout

Set a callback to execute when mouse leaves chart.

onmouseout: function() { ... }

onresize

Set a callback to execute when user resizes the screen.

onresize: function() { ... }

onresized

Set a callback to execute when screen resize finish.

onresized: function() { ... }

axis.rotated

Switch x and y axis position. Default is false.

axis: { rotated: true }

axis.x.show

Show or hides x axis. Default is true.

axis: { x: { show: true } }

axis.x.localtime

If true, treat x value as localtime. If false, convert to UTC internally. Default is true.

axis: { x: { localtime: true } }

axis.x.categories

Show ticks as categorized by each data.

axis: { x: { categories: ['a', 'b', 'c', 'd'] } }

axis.x.tick.centered

...

axis.x.tick.format

A function to format tick value. Format string is also available for timeseries data.

axis: { x: { tick: { format: function (x) { return x.getFullYear(); } } } }

axis.x.tick.culling

An object to configure culling ticks.

axis.x.tick.culling.max

The number of tick texts will be adjusted to less than this value.

axis: { x: { tick: { culling: { max: 5 } } } } }

axis.x.tick.count

The number of x axis ticks to show.

axis: { x: { tick: { count: 5 } } } }

axis.x.tick.fit

Fit x axis tick. Default is true.

axis: { x: { tick: { fit: true } } } }

axis.x.tick.values

...

axis.x.tick.rotate

Rotate x axis tick text. Default is 0. If you set negative value, it will rotate to opposite direction.

axis: { x: { tick: { rotate: 30 } } } }

axis.x.max

Set a max range for x axis. Range includes padding, set 0 if no padding needed.

axis: { x: { max: 200 } }

axis.x.min

Set a min range for x axis. Range includes padding, set 0 if no padding needed.

axis: { x: { min: -200 } }

axis.x.padding

Set padding for x axis.

axis: { x: { padding: { top: 200, bottom: 0 } } }

axis.x.height

...

axis.x.extent

Set default extent for subchart and zoom. This can be an array or function that returns an array.

axis: { x: { default: [5, 10]; } }

axis.x.label

Set label for axis.

axis.y.show

Show or hides y axis. Default is true.

axis: { y: { show: true } }

Whether to show each point in line. Default is true

point.r

The radius size of each point. Default is 2.5

{ point: { r: 5 } }

Whether to expand each point on focus. Default is true

{ point: { focus: { expand: { enabled: true } } } }

point.focus.expand.r

The radius size of each point on focus.

{ point: { focus: { expand: { r: 1 } } } }

point.select.r

The radius size of each point on selected.

{ point: { select: { r: 3 } } }
